PROJECTS
DETECTION OF ANIMALS USING CONVOLUTIONARY NEURAL NETWORKS

This intelligent system helps in reducing the roadkill during the active nocturnal hours. The system uses thermal image processing using the camera mount on the vehicle. The main method used to extract the features of the animals is VGG-16. The extracted features are given as input to the basic deep learning model, a one-dimensional convolutional network, where binary cross-entropy is used to detect the existence of the object.

OPTIMIZED PERFORMANCE MEDICAL IMAGE CLASSIFICATION WITH DEEP TREE CONVOLUTIONAL NEURAL NETWORKS

The performance of two popular convolutional neural network (CNN) architectures, xception-Net and VGG16, for medical image classification tasks. Specifically, we evaluate their performance on two different datasets: ISIC and CMNIST. For ISIC, we use xception-net to classify different stages of categories. For CMNIST data, we use VGG-16 to classify histology images for different stages of categories. The accuracy of Xception-net is 91.96% and accuracy of VGG-16 is 99.76%.
